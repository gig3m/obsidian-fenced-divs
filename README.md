# Pandoc Fenced Divs for Obsidian

Render [Pandoc fenced divs](https://pandoc.org/MANUAL.html#extension-fenced_divs) (`::: name ... :::`) as styled blocks in Obsidian. Source files stay as canonical Pandoc Markdown — any external converter (Pandoc, Typst, etc.) can pick them up unchanged.

This is the Obsidian preview surface only. PDF generation lives in a separate repo.

Full documentation will be filled in at v1.0.0 release.
